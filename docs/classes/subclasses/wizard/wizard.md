# 法师 Wizard

## 基础资料

- 职业 ID：`wizard`；规则集：`5e-2024`
- 主要属性：智力；生命骰：`d6`；豁免：智力、感知；子职：3级
- 技能：从奥秘、历史、洞悉、调查、医药、自然、宗教中选2项
- 训练：简易武器；无护甲；奥术法器或法术书
- 起始装备：A 2匕首、长棍奥术法器、长袍、法术书、学者套组、5 GP；B 55 GP
- 多职业前置：智力13

## 1级车卡与公式

1. 选择2项技能、3个戏法，并在法术书写入6个1环法术；从书中准备4个。
2. 法术书与准备列表必须分开保存；仪式精通可从书中施放带仪式标签且未准备的法术。

```text
1级最大生命值 = 6 + 体质调整值
法术攻击加值 = 熟练加值 + 智力调整值
法术豁免 DC = 8 + 熟练加值 + 智力调整值
```

## 资源与校验

- 升级后向法术书加入2个可施放环级的法师法术；抄录每法术环级耗2小时与50 GP。
- 准备法术长休可从法术书重选；戏法长休可替换1个。
- 奥术回想短休后恢复总环级不超过法师等级一半向上取整，且不能恢复6环以上槽，长休一次。
- 法术精通与招牌法术保存被选法术ID、替换时机和免费施放状态。
