# 武僧 Monk

## 基础资料

- 职业 ID：`monk`
- 规则集：`5e-2024`
- 主要属性：敏捷、感知；生命骰：`d8`
- 豁免熟练：力量、敏捷；子职选择：3级
- 技能：从杂技、运动、历史、洞悉、宗教、隐匿中选2项
- 武器：简易武器和具有轻型属性的军用武器；护甲：无
- 工具：选一种工匠工具或乐器
- 起始装备：A 长矛、5匕首、所选工具或乐器、探索套组、11 GP；B 50 GP
- 多职业前置：敏捷13且感知13

## 1级车卡与公式

1. 选择2项技能、1种工具或乐器和装备方案。
2. 武艺完整收益要求未穿甲、未持盾，且空手或仅持武僧武器。
3. 1级无专注点；2级启用疾风连击、坚忍防御和疾走如风。

```text
1级最大生命值 = 8 + 体质调整值
无甲防御 AC = 10 + 敏捷调整值 + 感知调整值
徒手/武僧武器攻击加值 = 熟练加值 + 敏捷调整值
武僧豁免 DC = 8 + 熟练加值 + 感知调整值
专注点上限 = 武僧等级（2级起）
```

## 资源与校验

- 专注点在短休或长休后全部恢复；超常代谢每长休一次，在先攻时恢复专注并回复生命。
- 10级高度专注覆盖三种基础专注动作的旧定义，不能重复叠加。
- 拨挡攻击用反应减伤，减至0后可消耗专注反击；13级扩展为拨挡能量。
- 震慑打击命中触发且每回合一次；无甲移动、武艺、无甲防御与护甲或盾牌互斥。
- 额外攻击不与其他职业同名能力简单相加，子职位唯一。
