# 牧师 Cleric

## 1. 文档范围

本文档记录牧师的基础职业资料、等级成长、车卡选择与项目实现边界。默认规则集为 `5e-2024`，公开规则依据 2024 Free Rules 与 SRD 5.2.1。

- 职业 ID：`cleric`
- 规则集：`5e-2024`
- 核心定位：使用感知施展神术，通过引导神力、治疗、防护与领域能力支援队伍或压制敌人
- 主要属性：感知
- 次要关注：体质影响生命值与专注豁免；力量或敏捷取决于护甲和武器路线
- 施法方式：完整施法者，使用感知，长休时可重新准备牧师法术
- 子职选择等级：3级
- 资料状态：基础职业与生命领域已按开放规则核验；商业子职只记录索引和简要转述

## 3. 1级车卡资料

### 3.1 必须完成的选择

1. 从历史、洞悉、医药、游说、宗教中选择2项不同技能熟练。
2. 选择起始装备方案 A 或 B；方案 A 需要记录圣徽的具体形态。
3. 从牧师法术列表选择3个不同戏法。
4. 从牧师法术列表选择4个1环法术作为准备法术。
5. 在神圣职分中选择守护者或奇术师。
6. 记录信仰对象、圣徽与教义等叙事信息；除团规则另有规定外，这些信息不额外改变数值。

### 3.2 神圣职分

| 稳定 ID | 中文名 | 英文名 | 效果摘要 |
|---|---|---|---|
| `cleric-divine-order-protector` | 守护者 | Protector | 获得军用武器熟练和重甲训练 |
| `cleric-divine-order-thaumaturge` | 奇术师 | Thaumaturge | 额外获得1个牧师戏法；智力（奥秘或宗教）检定获得等于感知调整值且最低为 `+1` 的加值 |

该选择必须持久化。守护者不会自动提供武器精通；奇术师的额外戏法不计入等级表的普通戏法数量。

### 3.3 1级派生值

```text
最大生命值 = 8 + 体质调整值
法术攻击加值 = 熟练加值 + 感知调整值
法术豁免 DC = 8 + 熟练加值 + 感知调整值
奇术师知识检定加值 = max(1, 感知调整值)
```

1级熟练加值为 `+2`，拥有3个戏法、4个准备法术和2个1环法术位。选择奇术师时实际拥有4个牧师戏法。

### 5.1 引导神力资源

- 2级获得2次；6级增至3次；18级增至4次。
- 短休恢复1次已消耗次数，长休恢复全部。
- 神圣火花、驱散亡灵及领域提供的引导神力选项共享同一资源池。
- 引导神力豁免 DC 等于牧师法术豁免 DC。

### 5.2 神圣火花

- 以魔法动作选择30尺内可见的另一生物。
- 掷 `1d8 + 感知调整值`，选择治疗等量生命值，或令目标进行体质豁免并承受黯蚀或光耀伤害；成功时伤害减半。
- 骰数在7、13、18级分别提升为 `2d8`、`3d8`、`4d8`。

## 7. 车卡校验要求

- 技能必须从职业允许列表选择2项并去重。
- 起始装备方案 A/B 互斥；重甲和军用武器必须具有守护者或其他明确来源。
- 神圣职分必须二选一；奇术师额外戏法与普通戏法分开记录。
- 普通准备法术数量、戏法数量和法术环阶不得超过等级表限制。
- 领域法术标记为始终准备，不占用普通准备法术数量。
- 法术攻击和豁免 DC 使用感知；法术位与准备法术必须分开保存。
- 引导神力的所有用途共享次数，短休只恢复1次而非全部。
- 3级以前不能取得领域机制；旧版领域必须记录规则集、来源、DM许可和等级映射。
- 2024神佑打击与旧版领域8级同类强化互斥，禁止能力叠加。
