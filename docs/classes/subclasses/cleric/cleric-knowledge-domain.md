# 知识领域 Knowledge Domain

## 基本信息

- 子职 ID：`cleric-knowledge-domain`
- 所属职业：[`cleric` 牧师](cleric.md)
- 规则集：`5e-2014`
- 原版选择等级：1级；套用2024基础职业时在3级取得原本1—3级的领域能力
- 内容来源：2014 Player's Handbook
- 可用范围：需 DM 允许该来源，并通过2024兼容性检查
- 官方详情：[D&D Beyond — 2014 Player's Handbook](https://www.dndbeyond.com/sources/dnd/phb-2014)

## 子职定位

知识领域强化语言、知识技能、临时熟练与信息获取。它能在需要时短暂掌握工具或技能，读取目标表层思想并施加影响，高等级还可从物品和地点追溯过去。

## 选择、资源与兼容

- 语言、知识技能和专精选项必须去重，并保留来源。
- 万古知识是临时熟练，不授予专精，也不能永久满足升级或制作前置。
- 读取思想消耗共享引导神力，需按目标、豁免、距离和接续效果分别校验。
- 往昔幻象需要区分物品解读与地点解读，并记录冷却。
- 套用2024牧师时，不再叠加旧版8级强力施法；7级按基础职业选择神佑打击。
