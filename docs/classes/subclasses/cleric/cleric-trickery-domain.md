## 子职定位

诡术领域通过潜行祝福和幻象分身改变战场位置。分身不再依赖专注，可作为施法位置与近战支援点；中高等级提供位置交换、攻击优势和结束时治疗。

## 选择、资源与校验

- 领域法术始终准备，不占普通准备法术数量。
- 诡术师祝福只维持一个目标，长休或重新指定时结束。
- 引出分身消耗引导神力和附赠动作，但2024版本不要求专注。
- 分身不是生物，必须使用独立位置实体，并限制可移动距离、施法起点和可交互能力。
- 换位必须满足当前距离与空间合法性，不能进入被占据或不可到达位置。
- 强化分身的攻击优势和结束治疗只在规则指定范围与时点结算。
