# 暮光领域 Twilight Domain

## 基本信息

- 子职 ID：`cleric-twilight-domain`
- 所属职业：[`cleric` 牧师](cleric.md)
- 规则集：`5e-2014`
- 原版选择等级：1级；套用2024基础职业时在3级取得原本1—3级的领域能力
- 内容来源：Tasha's Cauldron of Everything
- 可用范围：需 DM 允许该来源，并通过2024兼容性检查
- 官方详情：[D&D Beyond — Tasha's Cauldron of Everything](https://www.dndbeyond.com/sources/dnd/tcoe)

## 子职定位

暮光领域提供超远黑暗视觉、先攻支援和持续范围防护。暮光圣域每轮为队友提供临时生命值或清除特定状态，中等级可在昏暗环境飞行，高等级为圣域中的队友提供掩护。

## 选择、资源与兼容

- 军用武器和重甲训练与2024守护者重复时不叠加。
- 夜幕之眼的分享具有目标数、距离、持续时间和恢复限制，不能作为永久感官来源。
- 警觉祝福同时只能维持一个目标，触发先攻或重新指定后结束。
- 暮光圣域消耗共享引导神力；每个目标每轮在临时生命值和结束状态之间选择一个。
- 夜幕步伐依赖当前照明条件；条件、持续时间与着陆风险需要动态处理。
- 套用2024牧师时，不再叠加旧版8级神圣打击；7级按基础职业选择神佑打击。
