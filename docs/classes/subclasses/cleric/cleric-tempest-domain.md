# 风暴领域 Tempest Domain

## 基本信息

- 子职 ID：`cleric-tempest-domain`
- 所属职业：[`cleric` 牧师](cleric.md)
- 规则集：`5e-2014`
- 原版选择等级：1级；套用2024基础职业时在3级取得原本1—3级的领域能力
- 内容来源：2014 Player's Handbook
- 可用范围：需 DM 允许该来源，并通过2024兼容性检查
- 官方详情：[D&D Beyond — 2014 Player's Handbook](https://www.dndbeyond.com/sources/dnd/phb-2014)

## 子职定位

风暴领域专注雷鸣与闪电爆发。它以反应惩罚近身攻击者，用引导神力把雷鸣或闪电伤害最大化，闪电伤害还能推动敌人，最终获得户外飞行能力。

## 选择、资源与兼容

- 军用武器与重甲训练和2024守护者重复时不叠加。
- 风暴之怒消耗反应和独立次数，必须满足攻击者距离与伤害触发。
- 毁灭之怒消耗共享引导神力，最大化的是符合条件的伤害骰而非所有附带伤害。
- 雷霆打击只由闪电伤害触发，并受目标体型和空间限制。
- 套用2024牧师时，不再叠加旧版8级神圣打击；7级按基础职业选择神佑打击。
