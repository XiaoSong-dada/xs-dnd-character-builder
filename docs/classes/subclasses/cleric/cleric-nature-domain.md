# 自然领域 Nature Domain

## 基本信息

- 子职 ID：`cleric-nature-domain`
- 所属职业：[`cleric` 牧师](cleric.md)
- 规则集：`5e-2014`
- 原版选择等级：1级；套用2024基础职业时在3级取得原本1—3级的领域能力
- 内容来源：2014 Player's Handbook
- 可用范围：需 DM 允许该来源，并通过2024兼容性检查
- 官方详情：[D&D Beyond — 2014 Player's Handbook](https://www.dndbeyond.com/sources/dnd/phb-2014)

## 子职定位

自然领域把德鲁伊式自然魔法融入牧师。它提供额外戏法与技能、影响野兽和植物的引导神力，并能以反应为队友降低元素伤害，最终获得对受影响自然生物的指挥能力。

## 选择、资源与兼容

- 德鲁伊戏法仍以感知作为施法属性，但必须保留其领域来源。
- 重甲训练与2024守护者重复时不产生额外收益。
- 魅惑动植物按生物类型、范围、豁免和持续条件筛选目标。
- 抑制元素消耗反应，只适用于原特性列明的元素伤害类型。
- 套用2024牧师时，不再叠加旧版8级神圣打击；7级按基础职业选择神佑打击。
