# 圣武士 Paladin

## 基础资料

- 职业 ID：`paladin`；规则集：`5e-2024`
- 主要属性：力量、魅力；生命骰：`d10`
- 豁免熟练：感知、魅力；子职选择：3级
- 技能：从运动、洞悉、威吓、医药、说服、宗教中选2项
- 武器与护甲：简易与军用武器；轻甲、中甲、重甲、盾牌
- 起始装备：A 链甲、盾牌、长剑、6标枪、圣徽、祭司套组、9 GP；B 150 GP
- 多职业前置：力量13且魅力13

## 1级车卡与公式

1. 选择2项技能、2种武器精通、装备方案与2个准备法术。
2. 计算魅力施法值并建立圣疗池；圣疗与施法从1级启用，战斗风格和圣武士斩击从2级启用。

```text
1级最大生命值 = 10 + 体质调整值
法术攻击加值 = 熟练加值 + 魅力调整值
法术豁免 DC = 8 + 熟练加值 + 魅力调整值
圣疗池 = 5 × 圣武士等级
```

## 资源与校验

- 圣疗池长休恢复；奖励动作触碰治疗，消耗5点可移除中毒，不能同时治疗。
- 准备法术长休可更换；誓言法术和职业固定法术不占普通准备数。
- 引导神力3级起2次、11级起3次，短休恢复1次、长休全部恢复。
- 守护灵光以魅力调整值增加自身与范围内盟友豁免，不能由同名灵光重复叠加；18级扩大范围。
- 额外攻击不与其他职业同名特性相加；光耀打击每次武器/徒手命中按规则结算。
