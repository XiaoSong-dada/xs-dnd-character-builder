# 游侠 Ranger

## 基础资料

- 职业 ID：`ranger`；规则集：`5e-2024`
- 主要属性：敏捷、感知；生命骰：`d10`
- 豁免熟练：力量、敏捷；子职选择：3级
- 技能：从驯兽、运动、洞悉、调查、自然、察觉、隐匿、生存中选3项
- 武器与护甲：简易与军用武器；轻甲、中甲、盾牌
- 起始装备：A 镶钉皮甲、弯刀、短剑、长弓、20箭、箭袋、德鲁伊法器、探索套组、7 GP；B 150 GP
- 多职业前置：敏捷13且感知13

## 1级车卡与公式

1. 选择3项技能、2种武器精通、装备方案、2个准备法术。
2. 猎人印记始终准备，另有2次无槽施放；它不占普通准备法术数。

```text
1级最大生命值 = 10 + 体质调整值
法术攻击加值 = 熟练加值 + 感知调整值
法术豁免 DC = 8 + 熟练加值 + 感知调整值
```

## 资源与校验

- 法术位长休恢复，准备法术长休可替换；感知为施法属性。
- 宿敌提供猎人印记无槽施放，次数按1/5/9/13/17级增长为2/3/4/5/6，长休恢复。
- 猎人印记仍需要专注；无情猎手只保护该法术的专注，不移除专注规则。
- 不倦的临时生命不叠加；自然帷幕按奖励动作和长休次数管理。
- 额外攻击不与其他职业同名能力相加，子职选择位唯一。
