# 游荡者 Rogue

## 基础资料

- 职业 ID：`rogue`；规则集：`5e-2024`
- 主要属性：敏捷；生命骰：`d8`；豁免：敏捷、智力；子职：3级
- 技能：从杂技、运动、欺瞒、洞悉、威吓、调查、察觉、说服、巧手、隐匿中选4项
- 武器/工具/护甲：简易武器及具有灵巧或轻型属性的军用武器；盗贼工具；轻甲
- 起始装备：A 皮甲、2匕首、短剑、短弓、20箭、箭袋、盗贼工具、窃贼套组、8 GP；B 100 GP
- 多职业前置：敏捷13

## 1级车卡与公式

1. 选择4项技能，其中2项获得专精；选择2种已熟练武器的武器精通。
2. 保存盗贼黑话和额外语言；确认偷袭武器必须有灵巧或远程属性。

```text
1级最大生命值 = 8 + 体质调整值
武器攻击加值 = 熟练加值 + 敏捷调整值
偷袭伤害 = 职业表对应 d6
```

## 资源与校验

- 偷袭每回合一次，需优势或目标邻接合格盟友，且自身攻击不能处于劣势。
- 诡诈打击通过放弃偷袭骰支付选项成本；同一骰不能重复支付。
- 灵巧动作只提供疾走、撤离或躲藏；稳定瞄准要求本回合尚未移动并把速度变为0。
- 可靠才能只作用于已熟练的能力检定；直觉闪避消耗反应。
