# 吟游诗人 Bard

## 1. 文档范围

本文档记录吟游诗人的基础职业资料、等级成长、车卡选择与项目实现边界。默认规则集为 `5e-2024`，公开规则依据 2024 Free Rules 与 SRD 5.2.1。

- 职业 ID：`bard`
- 规则集：`5e-2024`
- 核心定位：以魅力施法、吟游激励、技能泛用性和魔法秘密承担支援、控制、交涉与知识角色
- 主要属性：魅力
- 次要关注：敏捷影响 AC、先攻与武器攻击；体质影响生命值与专注豁免
- 施法方式：完整施法者，使用魅力，按吟游诗人等级表准备法术
- 子职选择等级：3级
- 资料状态：基础职业与博闻学院已按开放规则核验；商业子职只记录索引和简要转述

## 3. 1级车卡资料

### 3.1 必须完成的选择

1. 从全部技能中选择3项不同技能熟练。
2. 选择3种不同的乐器工具熟练。
3. 选择起始装备方案 A 或 B；方案 A 还需选择1件乐器。
4. 选择2个吟游诗人戏法。
5. 选择4个1环吟游诗人法术作为准备法术。
6. 记录吟游激励次数，其上限等于魅力调整值，最低1次。

### 3.2 1级派生值

```text
最大生命值 = 8 + 体质调整值
法术攻击加值 = 熟练加值 + 魅力调整值
法术豁免 DC = 8 + 熟练加值 + 魅力调整值
吟游激励次数 = max(1, 魅力调整值)
```

1级熟练加值为 `+2`，吟游激励骰为 `d6`，拥有2个戏法、4个准备法术和2个1环法术位。

### 3.3 吟游激励的实现要点

- 激活：以附赠动作选择60尺内能看见或听见你的另一生物。
- 持有：目标同时只能持有1枚吟游激励骰，持续1小时。
- 使用：目标在一次 d20检定失败后掷激励骰并加入结果，可能把失败变为成功。
- 消耗：只有掷出激励骰时才消耗。
- 次数：等于魅力调整值，最低1次；1级在长休后全部恢复。
- 骰型：1级 `d6`、5级 `d8`、10级 `d10`、15级 `d12`。
- 5级灵感源泉使短休也能恢复全部次数，并允许消耗法术位恢复1次激励。

## 7. 车卡校验要求

- 3项技能和3种乐器都必须各自去重。
- 起始装备方案 A/B 互斥；方案 A 的乐器选择必须来自合法工具列表。
- 戏法、准备法术数量和法术环阶不得超过等级表限制。
- 吟游激励次数最低1次，目标不能同时持有多个激励骰。
- 激励骰只能在失败的d20检定后使用，并在掷出时消耗。
- 专精只能选择已有熟练的技能，同一来源不能重复选择同一技能。
- 法术攻击和豁免 DC 使用魅力；法术位与准备法术必须分开保存。
- 3级以前不能选择子职；旧版扩展子职必须记录规则集、来源和DM许可状态。
