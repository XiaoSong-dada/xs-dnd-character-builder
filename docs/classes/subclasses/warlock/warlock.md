# 邪术师 Warlock

## 基础资料

- 职业 ID：`warlock`；规则集：`5e-2024`
- 主要属性：魅力；生命骰：`d8`；豁免：感知、魅力；子职：3级
- 技能：从奥秘、欺瞒、历史、威吓、调查、自然、宗教中选2项
- 训练：简易武器、轻甲、奥术法器
- 起始装备：A 皮甲、镰刀、2匕首、宝珠奥术法器、秘学书、学者套组、15 GP；B 100 GP
- 多职业前置：魅力13

## 1级车卡与公式

1. 选择2项技能、2个戏法、2个1环准备法术和1项魔能祈唤。
2. 契约魔法1级只有1个1环槽；2级获得魔法机巧并把祈唤增至3项。

```text
1级最大生命值 = 8 + 体质调整值
法术攻击加值 = 熟练加值 + 魅力调整值
法术豁免 DC = 8 + 熟练加值 + 魅力调整值
```

## 资源与校验

- 契约魔法槽全为表中同一环级，短休或长休恢复；与普通施法槽必须分池保存。
- 魔法机巧需1分钟，长休一次，恢复不超过最大槽数一半向上取整。
- 魔能祈唤校验等级、契约祈唤和依赖关系；作为其他祈唤前置的项目不能被替换。
- 秘法奥秘6—9环各独立长休一次，不是契约魔法槽。
