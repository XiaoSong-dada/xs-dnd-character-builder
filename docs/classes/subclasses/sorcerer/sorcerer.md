# 术士 Sorcerer

## 基础资料

- 职业 ID：`sorcerer`；规则集：`5e-2024`
- 主要属性：魅力；生命骰：`d6`；豁免：体质、魅力；子职：3级
- 技能：从奥秘、欺瞒、洞悉、威吓、说服、宗教中选2项
- 武器/工具/护甲：简易武器；无护甲训练；奥术法器
- 起始装备：A 长矛、2匕首、水晶奥术法器、地城探险套组、28 GP；B 50 GP
- 多职业前置：魅力13

## 1级车卡与公式

1. 选择2项技能、4个戏法、2个1环准备法术和装备。
2. 天生术法每天2次；2级获得术法点与2项超魔法。

```text
1级最大生命值 = 6 + 体质调整值
法术攻击加值 = 熟练加值 + 魅力调整值
法术豁免 DC = 8 + 熟练加值 + 魅力调整值
术法点上限 = 术士等级（2级起）
```

## 资源与校验

- 天生术法奖励动作开启1分钟：术士法术DC+1且法术攻击优势；长休恢复。
- 术法点长休恢复；可与1—5环法术位互换，转换成本必须使用官方表。
- 超魔法必须校验法术资格、每次施法可用数量和术法点成本。
- 子职常备法术不占普通准备数；升级时普通准备法术的替换遵守等级限制。
