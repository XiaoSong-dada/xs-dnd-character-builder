# 战士 Fighter

## 基础资料

- 职业 ID：`fighter`
- 规则集：`5e-2024`
- 主要属性：力量或敏捷；生命骰：`d10`
- 豁免熟练：力量、体质；子职选择：3级
- 技能：从杂技、驯兽、运动、历史、洞悉、威吓、察觉、说服、生存中选2项
- 武器：简易、军用；护甲：轻甲、中甲、重甲、盾牌
- 起始装备：A 链甲、巨剑、连枷、8标枪、地城探险套组、4 GP；B 镶钉皮甲、弯刀、短剑、长弓、20箭、箭袋、地城探险套组、11 GP；C 155 GP
- 多职业前置：力量13或敏捷13

## 1级车卡与公式

1. 选择力量近战、敏捷远程或灵巧武器路线。
2. 选择2项技能、1项战斗风格专长、3种武器精通和装备方案。
3. 保存每个精通的武器 ID；盾牌、双手武器与施法占手必须校验。

```text
1级最大生命值 = 10 + 体质调整值
武器攻击加值 = 熟练加值 + 对应力量或敏捷调整值
回气治疗 = 1d10 + 战士等级
```

## 资源与校验

- 回气：奖励动作治疗；短休恢复1次，长休全部恢复。战术心智会在失败检定后暂用1次，若加骰后仍失败则不消耗。
- 动作如潮：短休或长休恢复；额外动作不能用于魔法动作。
- 不屈：失败豁免后重投并加入战士等级，必须使用新结果。
- 额外攻击按5/11/20级变为2/3/4击，不同职业的同名特性不能简单相加。
- 校验战斗风格资格、武器精通数量、回气次数、动作限制与唯一子职位。
